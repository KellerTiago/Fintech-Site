# Fintech-Site

Para usuários que queiram organizar os diversos aspectos da vida financeira de modo compartilhado.

Que estão insatisfeitos com as alternativas de inter-relação no processo de gestão financeira.

Nosso produto Financial Project é uma aplicação financeira que permite o planejamento, organização de gastos, rendimentos e investimentos de forma colaborativa entre determinados usuários.

Que fornece a capacidade de tratar de forma conjunta as finanças de uma família ou até mesmo de pessoas sem grau de parentesco, mediante ao cadastro de um projeto e seus usuários.

Ao contrário de outras aplicações onde o gerenciamento financeiro ocorre de um modo mais individualizado.

Nosso produto contribui para uma gestão financeira interativa, transparente e com foco na colaboração para obtenção de objetivos em comum.
